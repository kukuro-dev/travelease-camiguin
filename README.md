#  TravelEase Camiguin

A web-based tourism platform designed to connect tourists and local providers in Camiguin Island. TravelEase Camiguin allows users to explore tourist destinations, discover accommodations and rental services, communicate with local businesses, and access travel assistance through an interactive and user-friendly platform.

---

##  Project Description

TravelEase Camiguin is a tourism-focused web application developed to promote local tourism by providing a digital platform where tourists can discover destinations, accommodations, and activities while allowing local providers to showcase their services.

The system creates a connection between tourists and local businesses through features such as messaging, reviews, service listings, and an AI-powered travel assistant.

The platform supports various tourism services, including:

*  Tourist destinations
*  Cottages and resorts
*  Snorkeling equipment rentals
*  Bike rentals
*  Motorcycle rentals

#  Installation Guide

## 1. Clone the Repository

```bash
git clone https://github.com/kukuro-dev/travelease-camiguin.git
```

## 2. Move Project to XAMPP

Copy the project folder into:

```text
C:\xampp\htdocs\camiguin-rental
```

## 3. Start XAMPP

Open XAMPP Control Panel and start:

* Apache
* MySQL

## 4. Setup Database

1. Open phpMyAdmin:

```text
http://localhost/phpmyadmin
```

2. Create a new database.

3. Import the SQL database file included in the project.

Example:

```text
database/travelease_camiguin.sql
```

## 5. Configure Database Connection

Update your database configuration file:

```php
Host: localhost
Username: root
Password:
Database: travelease_camiguin
```

## 6. Run the Application

Open your browser:

```text
http://localhost/camiguin-rental
```
